Edison SMS Alerts

Edison is a personal hobby project that sends SMS alerts related to personal project updates, device events, test notifications, and system status updates.

How to Opt In

To opt in, text START to +1 (612) 260-8909.

By texting START to this number, the user confirms they want to receive SMS messages from Edison. Texting this phone number is the only way to opt in to messaging.

This program is intended only for the project owner or explicitly approved testers.

Program Details

  • Program name: Edison
  • Use case: personal hobby project alerts and testing
  • Message frequency: varies based on project activity and testing
  • Pricing disclosure: message and data rates may apply
  • Opt out: reply STOP at any time
  • Help: reply HELP or contact dwashin@outlook.com

Sample Messages

Opt-in confirmation:
Edison: You are subscribed to personal project alerts and test messages. Message frequency varies. Message and data rates may apply. Reply HELP for help, STOP to opt out.

Sample alert:
Edison: Device status update available. Reply STOP to opt out, HELP for help.
